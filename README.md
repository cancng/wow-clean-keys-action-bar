# Clean Keys ActionBar

Makes keybind text on action bars cleaner and more readable.

## Preview

![Keybind Preview](https://nbg1.your-objectstorage.com/bomi/common/wow-addons/ckb/sample.png)

## Settings

Open settings with `/ck` or `/cleankeys` command.

![Settings](https://nbg1.your-objectstorage.com/bomi/common/wow-addons/ckb/settings.png)

## Key Conversions

| Original | Clean |
|----------|-------|
| CTRL-BUTTON4 | CM4 |
| SHIFT-ALT-5 | SA5 |
| MOUSEWHEELUP | MWU |
| NUMPAD1 | N1 |

## Modifiers

- **C** = CTRL
- **S** = SHIFT
- **A** = ALT
